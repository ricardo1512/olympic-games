# olympic-games
The "One Century of Olympic Games" is a web application developed by Ricardo Vicente in 2024, aimed at providing insights into Olympic Games data from 1896 to 2016. It supports modern browsers and is built using JavaScript, HTML, and CSS. The application features various interactive tools and visualizations, including a Choropleth Map, Cleveland Dot Plot, Line Chart, and Scatter Plot. Users can explore data over time using a timeline with a brush, and apply filters to view information by country and season (Winter and Summer). This platform allows for an interactive exploration and analysis of Olympic Games data, offering a detailed view of historical trends and performances across different dimensions, such as continent, gender, sports, and GDP per capita.
